# اگر هدفت واقعاً "دیپ" است

از سوال‌هایی که این چند روز پرسیدی (JVM، Boxing، Stack Frame، Memory، Call Stack) به نظرم تو از جنس کسی هستی که دوست داره **بدونه چرا**، نه فقط **چطور**.

برای همین این موضوعات رو به همین ترتیب جلو برو:

1. Call Stack
2. Stack Frame
3. Heap
4. Method Area
5. Bytecode
6. JVM Execution Engine
7. ClassLoader
8. Garbage Collection
9. Java Memory Model
10. JIT Compiler

وقتی این‌ها را یاد بگیری، جاوا برایت فقط یک زبان برنامه‌نویسی نیست؛ می‌فهمی JVM واقعاً چگونه برنامه را اجرا می‌کند.

**یک نکته هم بگم:** خیلی از ویدیوهای یوتیوب سطحی هستند. برای موضوعات عمیق، معمولاً عبارت‌های **"internals"**، **"under the hood"**، **"deep dive"** یا **"how it works"** را به انتهای جستجو اضافه کن. این کار معمولاً نتایج بسیار باکیفیت‌تری نسبت به جستجوی ساده مثل "Java Stack" یا "JavaScript Closures" می‌دهد.
