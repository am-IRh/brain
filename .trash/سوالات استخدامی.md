#  Help Desk (Microsoft Environment)  
  
- چطور یه نرم افزار رو در ویندوز نصب یا حذف می‌کنی  
برای نصب اول فایل setup یا installer رو اجرا می‌کنیم با پسوند (.msi .exe )  
برای حذف هم از مسیرcontrol panel -> programs and features حذف می‌کنیم  
گاهی هم از PowerShell با دستور

Get-WmiObject

یا

winget uninstall

استفاده می‌شه.

- اگه یه نرم‌افزار باز نمی‌شه یا ارور می‌ده، چطور بررسی می‌کنی؟  
اول چک می‌کنم سیستم به‌روزه و فایل برنامه سالمه.  
بعد توی Task Manager بررسی می‌کنم که برنامه اجرا شده یا نه.  
اگه هنوز باز نشد، با Event Viewer لاگ خطاها رو بررسی می‌کنم.  
در نهایت اگه لازم باشه reinstall می‌کنم یا با نسخه قبلی جایگزین.  
📌 نکته: توی مصاحبه اگر بگی **Event Viewer**، مصاحبه‌گر سریع متوجه می‌شه با مفاهیم پایه آشنا هستی.  
  
- : چطور سرعت پایین سیستم رو بررسی و رفع می‌کنی؟  
با Task Manager شروع می‌کنم تا ببینم CPU، RAM یا Disk زیادی مصرف میشه یا نه.  
برنامه‌های غیرضروری Startup رو غیرفعال می‌کنم.  
سیستم رو از نظر ویروس با Windows Defender یا آنتی‌ویروس بررسی می‌کنم.  
و در صورت نیاز دیسک رو پاکسازی (Disk Cleanup) یا Defragment می‌کنم.  
📌 ابزارهایی که می‌تونی اسم ببری: `Task Manager`, `msconfig`, `Disk Cleanup`, `Windows Defender`.  
  
- تفاوت Local Account و Domain Account چیه؟  
ع Local Account فقط روی همون سیستم قابل استفاده‌ست، ولی Domain Account از طریق سرور Active Directory کنترل میشه و کاربر می‌تونه با اون توی هر سیستم عضو Domain وارد بشه.  
  
- اگه کاربر نتونه وارد سیستم بشه و ارور Password اشتباه بده، چی کار می‌کنی؟  
اول مطمئن می‌شم کیبورد درست کار می‌کنه و Caps Lock روشن نیست.  
بعد بررسی می‌کنم کاربر در Domain قفل نشده باشه (Account Locked).  
در صورت نیاز از طریق Active Directory رمز رو ریست می‌کنم یا کاربر رو Unlock می‌کنم.  
  
📌 نکته: گفتن “check Caps Lock” یا “account lockout” نشون میده منطقی فکر می‌کنی.  
  
## Networking Basics  
-  اگه کاربر بگه اینترنت نداره، از کجا شروع می‌کنی؟  
اول بررسی می‌کنم کابل شبکه یا وای‌فای وصله یا نه.  
بعد با `ping 8.8.8.8` تست می‌کنم که اینترنت داریم یا نه.  
اگه ping جواب بده ولی سایت باز نشه، احتمالاً DNS مشکل داره.  
در اون صورت `ipconfig /flushdns` یا تغییر DNS به** 8.8.8.8 **انجام می‌دم.  
در نهایت تنظیمات IP رو چک می‌کنم (Static یا DHCP).  
  
- و IP Address و Gateway و DNS رو چطور توضیح می‌دی؟  
خب IP آدرس مخصوص هر دستگاه در شبکه‌ست.  
و Gateway مسیریه که دستگاه از اون برای خروج از شبکه خودش به اینترنت استفاده می‌کنه.  
و DNS سرویسیه که اسم دامنه (مثل** google.com**) رو به IP تبدیل می‌کنه.  
  
- دستور `ipconfig` چه کاری انجام میده؟  
اطلاعات مربوط به کارت شبکه، مثل IP Address، Subnet Mask و Default Gateway رو نشون میده.   
با `ipconfig /release` و `ipconfig /renew` هم می‌تونیم IP جدید بگیریم.  
  
- و VPN چیه و برای چی استفاده میشه؟  
تونلیه که ارتباط بین کاربر و شبکه سازمانی رو به‌صورت امن برقرار می‌کنه.   
مثلاً کارمند می‌تونه از خونه به منابع شرکت وصل بشه انگار داخل شرکت هست.  
  
### 💾 بخش سوم: سخت‌افزار و عیب‌یابی  
  
- اگه سیستم روشن نمی‌شه، چی بررسی می‌کنی؟  
اول بررسی می‌کنم برق وصله یا نه.  
بعد کابل‌ها و دکمه پاور رو چک می‌کنم.  
اگه فن‌ها روشن می‌شن ولی تصویر نیست، ممکنه RAM یا کارت گرافیک مشکل داشته باشه.  
با تست کردن قطعات یکی‌یکی می‌تونم مشکل رو پیدا کنم.  
📌 نکته: لازم نیست جواب خیلی فنی بدی، مهمه “منظم و منطقی” بررسی کنی.  
  
- تفاوت HDD و SSD چیه؟  
از دیسک مکانیکی استفاده می‌کنه و سرعت پایین‌تری داره،   
ولی SSD از حافظه فلش استفاده می‌کنه و خیلی سریع‌تر و کم‌صداست.  
  
- : اگه پرینتر کار نکنه، چطور بررسی می‌کنی؟  
اول بررسی می‌کنم پرینتر روشن و به شبکه یا کابل وصل باشه.  
بعد توی Control Panel → Devices and Printers می‌بینم که پرینتر Default درسته یا نه.  
اگه ارور میده، درایورش رو reinstall می‌کنم یا Print Spooler Service رو restart می‌کنم.  
  
اگه کاربر نتونه ایمیل بفرسته چی کار می‌کنی؟  
بررسی می‌کنم اینترنت فعاله یا نه.  
بعد تنظیمات سرور SMTP و Portها رو در Outlook چک می‌کنم.  
اگه درست بود، cache Outlook یا profile جدید می‌سازم.  
  
- تفاوت CC و BCC در ایمیل چیه؟  
CC برای کسانیه که باید ایمیل رو ببینن ولی مخاطب اصلی نیستن.  
BCC هم مثل CCه ولی گیرنده‌ها نمی‌تونن ببینن چه کسانی در BCC هستن.  
  
- و OneDrive چیه؟  
یه سرویس ابری مایکروسافته برای ذخیره و همگام‌سازی فایل‌ها بین دستگاه‌ها.   
مثلاً کاربر می‌تونه فایلش رو روی OneDrive ذخیره کنه و از هرجای دیگه بهش دسترسی داشته باشه.
